¿Qué es y qué hace? La práctica implementa un modelo de Aprendizaje Automático capaz de clasificar un mensaje de texto entrante como "ham" (mensaje legítimo) o "spam" (mensaje no deseado). Para lograr esto, se transforma el texto de los mensajes en características numéricas que el modelo puede interpretar.
¿Qué aprendimos? Aprendimos la metodología básica de un proyecto de Machine Learning basado en texto, que incluye: Procesamiento de Lenguaje Natural: Entender cómo el texto debe ser preprocesado (limpieza, tokenización). Representación de Texto: Utilizar técnicas para convertir palabras en vectores numéricos ponderados.
Lenguaje de Programación: Python. Análisis de Datos: Pandas (para la manipulación y lectura de los datos CSV). Machine Learning / NLP: Scikit-learn (sklearn). Esta es la librería central.
Mejora de Modelos: Probar clasificadores más avanzados como SVM o Redes Neuronales (RNN/LSTM) para manejar mejor la complejidad del lenguaje. Realizar ajuste fino de parámetros (Hyperparameter Tuning) para optimizar el rendimiento del modelo Naive Bayes actual y del vectorizador TF-IDF.Refinamiento del Procesamiento de Texto (NLP): Implementar la eliminación de stop words y la Lemmatización para limpiar aún más los datos y mejorar la precisión del modelo. Despliegue de la Solución: El paso final sería desplegar el modelo (usando herramientas como Flask) para crear una aplicación funcional donde los usuarios puedan probar el detector de spam en tiempo real.
